Set up the Tenant Account

  1. Open a browser and connect to the CCM using the URL provided by CliQr (for CliQr-Hosted SaaS Deployments) or the IP address of your  CCM server (for Customer-Hosted Deployments).
  2. Login as the CliQr platform admin. A platform admin can setup payment controls, register the cloud, and Tenants and Users.
  3. Create a license plan for the admin, by clicking the Admin link displayed at the top right corner of the CCM UI.
  4. Assign a plan or a bundle (contract) based on your contract agreement with CliQr. See Financial Enforcement for additional details.


  5. Assign a plan or contract to the CloudCenter platform admin, by clicking the  Users tab in the left pane and accessing the User Management page.
  6. Configure the cloud and enable the CCOs for managed clouds.
  7. Add the cloud information for your enterprise and Register the CCO with the CCM.
  8. Configure users by first creating each user and then assigning them to groups or roles. See User Management for additional details.
  9. Map pre-existing images or services. See Map Images for additional details.
  10. Create the required storage for the root tenant. See Deploy Storage for additional details.

Onboard the Application

This tour uses the Topology Builder to onboard applications.

  1. Define a new app.
  2. Add details for the selected application.
  3. Define the Global Parameters .
  4. Define the application architecture and components for this application.
  5. Save the application as an Application Profile.
  6. Select the cloud to deploy the application.

Deploy the Application

  1. Run the application.
  2. Manage you application deployment environment.
  3. Implement policies.

Optimize Performance

  1. Generate and analyze reports.
  2. Monitor the cloud health and status using dashboards.
  3. Benchmark your application.
